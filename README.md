# Demo-Notebooks

This repository combines the general [JAAT](https://github.com/Job-Ad-Research-at-QSB-LUC/JAAT) demo notebook with the public-source demo collection from [pnorlander/JAAT_Demos](https://github.com/pnorlander/JAAT_Demos).

## Demos

| Demo | Description |
| --- | --- |
| [JAAT Basics Demo](jaat_basics_demo.ipynb) | General JAAT walkthrough imported from the main JAAT repository. |
| [OSHA Accident Narratives](osha_accident_taskmatch/OSHA_TaskMatch_Demo.ipynb) | TaskMatch demo using OSHA accident investigation narratives. |
| [TitleMatch Validation](titlematch_validation/TitleMatch_Validation.ipynb) | TitleMatch validation workflow using public LCA title-code pairs. |
| [AI Curriculum Analysis](ai_curriculum/AI_Curriculum_JAAT_Analysis.ipynb) | JAAT analysis of public AI course syllabi. |
